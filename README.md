# Collection of .files

These configuration files contain a modern Bash & Vim setup with some git and docker aliases. It also has some various configuration files for macOS. To link these files, I recommend using [stow](http://www.gnu.org/software/stow/).

```shell
$ git clone --recurse-submodules -j8 git@github.com:deybhayden/DotFiles.git
$ cd DotFiles
$ ./macsimus.sh
```

## Installation

### macOS

The `macsimus.sh` script can be used to install a base system setup on a Mac. It can also be used to keep the Homebrew configuration up to date.
